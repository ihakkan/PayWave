# **App Name**: PayWave

## Core Features:

- Profile Header: Displays user profile icon, search bar, and notification bell.
- Main Services Grid: Presents core services such as 'Scan & Pay' and 'Send Money' using a grid of clickable icons.
- Promotional Banner: A visually appealing banner to display current offers.
- Recent Transactions List: Shows a list of recent transactions with details like recipient, amount, and date.
- Bottom Navigation: A fixed bottom navigation bar for easy access to 'Home', 'Wallet', 'Offers', and 'Profile'.

## Style Guidelines:

- Primary color: Dark blue (#24305E) to give the app a professional, trustworthy, and calm feeling. 
- Background color: Very light blue (#E0E5F3) to complement the primary color and maintain a light theme.
- Accent color: Light purple (#827397) to draw attention to interactive elements and provide visual interest without overwhelming the design.
- Font: 'Inter', a sans-serif font, for both headers and body text. It provides a clean and modern look.
- Use lucide-react icons for a clear and consistent visual language.
- Mobile-first, responsive design with a single-file React component structure.
- Subtle animations and transitions for interactive elements to enhance user experience.
